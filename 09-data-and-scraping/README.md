# Data & Scraping

Curated collection of the best open-source tools for data collection, processing, and scraping.

## Subcategories

| Folder | What It Covers |
|--------|---------------|
| `web-scrapers/` | General-purpose scrapers, browser automation |
| `api-wrappers/` | Pre-built integrations for popular services |
| `data-pipelines/` | ETL, data transformation, scheduling |
| `databases/` | Self-hosted DB tools, ORMs, migration tools |
| `visualization/` | Dashboard builders, chart libraries, reporting |
| `ai-data-tools/` | Dataset builders, labeling, cleaning |

## Selection Criteria
- Stars: >500 minimum, >1000 preferred
- Actively maintained (commits within 6 months)
- Complete start-to-finish solutions
- Clear documentation and setup guides
- Permissive license (MIT, Apache 2.0)
