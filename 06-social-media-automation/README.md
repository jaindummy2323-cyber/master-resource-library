# Social Media Automation

Curated collection of the best open-source tools for social media automation.

## Subcategories

| Folder | What It Covers |
|--------|---------------|
| `instagram/` | Posting, stories, reels automation, growth tools, analytics |
| `twitter-x/` | Tweet schedulers, thread builders, engagement tools |
| `linkedin/` | Connection automation, content posting, lead gen |
| `tiktok/` | Video posting, trend analysis, growth automation |
| `multi-platform/` | Cross-posting tools, unified dashboards |
| `analytics/` | Social listening, sentiment analysis, reporting |

## Selection Criteria
- Stars: >500 minimum, >1000 preferred
- Actively maintained (commits within 6 months)
- Complete start-to-finish solutions
- Clear documentation and setup guides
- Permissive license (MIT, Apache 2.0)
