# Marketing & SEO

Curated collection of the best open-source tools for marketing and SEO.

## Subcategories

| Folder | What It Covers |
|--------|---------------|
| `seo-audit-tools/` | Site crawlers, technical SEO, performance |
| `keyword-research/` | Search volume, competition, opportunity finder |
| `backlink-tools/` | Link building, analysis, outreach |
| `email-marketing/` | Campaign builders, list management, templates |
| `funnel-builders/` | Sales funnels, landing pages, A/B testing |
| `advertising/` | Ad copy generators, campaign management, budget optimization |

## Selection Criteria
- Stars: >500 minimum, >1000 preferred
- Actively maintained (commits within 6 months)
- Complete start-to-finish solutions
- Clear documentation and setup guides
- Permissive license (MIT, Apache 2.0)
