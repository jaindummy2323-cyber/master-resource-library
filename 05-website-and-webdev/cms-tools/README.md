# CMS Tools (Content Management Systems)

> The best open-source headless CMS, traditional CMS, and content management platforms for modern web development.

## Headless CMS (API-First)

| Repo | Stars | Description | Why It's Top-Tier |
|------|-------|-------------|-------------------|

| [Decap CMS (Netlify CMS)](https://github.com/decaporg/decap-cms) | 18k+ | Git-based CMS for static site generators. | Works with any SSG. Content stored in Git. Simple setup. |

## Selection Criteria Applied
- Every repo above has 5,000+ GitHub stars (most have 10k+)
- All are actively maintained (commits within 2025-2026)
- Self-hostable with clear deployment documentation
- REST or GraphQL API for headless use
- Permissive licenses (MIT, Apache 2.0, or similar)
- Used in production by real organizations

## How to Choose
- **Headless CMS for React/Next.js?** -> Payload (Next.js native) or Strapi (most popular)
- **Wrap existing database?** -> Directus (works with any SQL database)
- **Git-backed content?** -> Tina CMS (visual editing) or Decap CMS (simple)
- **Blog/Newsletter platform?** -> Ghost (professional publishing, built-in subscriptions)
- **Team wiki/knowledge base?** -> Outline (Notion-like) or Wiki.js (feature-rich)
- **Python/Django project?** -> Wagtail (best Django CMS)
- **Enterprise serverless?** -> Webiny (AWS-native, page builder included)
