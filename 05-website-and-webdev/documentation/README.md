## Why These Are Top-Tier

- **Docusaurus** — Battle-tested by Meta, React ecosystem standard, 57k+ stars
- **MkDocs Material** — Best Python docs experience, used by FastAPI, Pydantic, thousands more
- **Storybook** — Industry standard for component documentation, 85k+ stars
- **Mermaid** — Diagrams-as-code integrated into GitHub, Notion, Confluence natively
- **Scalar** — Modern API docs that actually look good and work well
- **Outline** — Best self-hosted knowledge base for teams

## Getting Started

| Use Case | Recommended Stack |
|---|---|
| Developer documentation | Docusaurus or VitePress |
| Python project docs | MkDocs Material |
| API reference | Scalar + OpenAPI spec |
| Component library docs | Storybook + MDX |
| Team knowledge base | Outline or Wiki.js |
| Changelogs | Changesets + Semantic Release |
| Architecture diagrams | Mermaid or D2 |
