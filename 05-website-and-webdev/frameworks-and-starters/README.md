# Frameworks & Full-Stack Starters

The foundation of every web project. These are the top frameworks and production-ready boilerplates used by millions of developers worldwide.

---

## Frameworks (Official)

| Repo | Stars | Description | Why It's Top-Tier |
|------|-------|-------------|-------------------|
| [Next.js](https://github.com/vercel/next.js) | 130k+ | React framework by Vercel — SSR, SSG, API routes, App Router | **The** full-stack React framework. Used by Netflix, TikTok, Hulu. Official Vercel product. |
| [React](https://github.com/facebook/react) | 230k+ | UI library by Meta — component-based, virtual DOM | The foundation everything else builds on. Official Meta product. |
| [Vue.js](https://github.com/vuejs/core) | 48k+ | Progressive JavaScript framework — reactive, approachable | Gentler learning curve than React. Massive ecosystem. |
| [Svelte](https://github.com/sveltejs/svelte) | 80k+ | Compile-time framework — no virtual DOM, minimal runtime | Fastest framework. Write less code, get better performance. |
| [Astro](https://github.com/withastro/astro) | 48k+ | Content-first framework — islands architecture, zero JS by default | Best for content sites. Ship HTML, add JS only where needed. |
| [Angular](https://github.com/angular/angular) | 97k+ | Full platform by Google — TypeScript-first, batteries included | Enterprise standard. Official Google product. |
| [Remix](https://github.com/remix-run/remix) | 30k+ | Full-stack React framework — web standards, nested routing | Built by React Router creators. Now part of Shopify. |
| [Nuxt](https://github.com/nuxt/nuxt) | 55k+ | Vue meta-framework — SSR, SSG, API, file-based routing | The Next.js of the Vue world. Production-ready. |
| [SvelteKit](https://github.com/sveltejs/kit) | 18k+ | Svelte meta-framework — SSR, routing, adapters | Official Svelte framework for full apps. |
| [SolidJS](https://github.com/solidjs/solid) | 33k+ | Reactive UI library — fine-grained reactivity, no virtual DOM | Fastest reactive framework. True signals architecture. |

## Build Tools & Bundlers

| Repo | Stars | Description | Why It's Top-Tier |
|------|-------|-------------|-------------------|
| [Vite](https://github.com/vitejs/vite) | 70k+ | Next-gen build tool — instant HMR, Rollup-based | Default build tool for Vue, Svelte, and new React projects. |
| [Turbo (Turborepo)](https://github.com/vercel/turborepo) | 26k+ | Monorepo build system — caching, parallel execution | Official Vercel. Best for multi-package repos. |
| [Bun](https://github.com/oven-sh/bun) | 75k+ | JavaScript runtime + bundler + package manager — all-in-one | Fastest JS runtime. Drop-in Node.js replacement. |
| [esbuild](https://github.com/evanw/esbuild) | 38k+ | Extremely fast JavaScript bundler — written in Go | 10-100x faster than webpack. Used inside Vite. |
| [Biome](https://github.com/biomejs/biome) | 16k+ | Fast formatter + linter — Rust-based, replaces ESLint + Prettier | One tool replaces two. Written in Rust for speed. |

## Selection Criteria Applied
- Every repo above has 5,000+ GitHub stars minimum
- All are actively maintained (commits within 2025-2026)
- Complete frameworks or production-ready starters
- Clear documentation with getting-started guides
- Permissive licenses (MIT, Apache 2.0)
- Used in real production deployments
