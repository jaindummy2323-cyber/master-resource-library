## Implementation Guide

| Use Case | Recommended Stack |
|---|---|
| Full notification system | Novu (all channels) |
| Transactional email | Nodemailer + React Email + Resend/SES |
| Email templates | React Email or MJML + Maizzle |
| In-app notifications | Novu In-App or Sonner (toasts) |
| Web push | web-push + Service Worker + Ntfy |
| Newsletter | Listmonk (self-hosted) |
| SMS | Twilio or Fonoster (self-hosted) |
| Webhooks | Svix or Convoy |
| Development testing | MailHog (catches all email locally) |
