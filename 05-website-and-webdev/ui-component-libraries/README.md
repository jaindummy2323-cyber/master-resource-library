## Selection Criteria Applied
- Every repo above has 2,000+ GitHub stars (most have 10k+)
- All are actively maintained (commits within 2025-2026)
- Complete component systems, not individual widgets
- Clear documentation with live examples
- Permissive licenses (MIT, Apache 2.0)
- Used in production by real companies

## How to Choose
- **Starting fresh with React?** → shadcn/ui + Radix (modern, you own the code)
- **Enterprise React app?** → MUI or Ant Design (battle-tested at scale)
- **Best developer experience?** → Chakra UI or Mantine (great DX, accessible)
- **Vue project?** → Vuetify (Material) or Element Plus (enterprise)
- **Svelte project?** → Skeleton or shadcn-svelte
- **Framework-agnostic?** → DaisyUI (just CSS classes on Tailwind)
- **Dashboard/Analytics?** → Tremor (chart-focused React components)
- **Full control over styling?** → Radix UI or Headless UI (unstyled primitives)
