## Selection Criteria Applied
- Every repo above has 5,000+ GitHub stars (most have 10k+)
- All are actively maintained (commits within 2025-2026)
- Production-ready with clear documentation
- Support major databases (Postgres, MySQL, SQLite at minimum)
- Permissive licenses (MIT, Apache 2.0)

## How to Choose
- **TypeScript ORM (modern)?** -> Drizzle (SQL-like, lightweight) or Prisma (auto-generated types)
- **TypeScript ORM (mature)?** -> TypeORM (Active Record) or Sequelize (established)
- **Python ORM?** -> SQLAlchemy (standard) or SQLModel (FastAPI companion)
- **Complete backend platform?** -> Supabase (Postgres) or PocketBase (SQLite, single binary)
- **Airtable replacement?** -> NocoDB (turns DB into spreadsheet) or Baserow (no-code)
- **Database GUI?** -> DBeaver (universal) or Beekeeper Studio (modern, beautiful)
- **Object storage?** -> MinIO (S3-compatible, self-hosted)
