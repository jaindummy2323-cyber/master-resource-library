## Selection Criteria Applied
- Every repo above has 3,000+ GitHub stars (most have 10k+)
- All are actively maintained (commits within 2025-2026)
- Production-ready libraries used in real applications
- Clear documentation with examples and demos
- Permissive licenses (MIT, Apache 2.0)

## How to Choose
- **Utility-first CSS?** → Tailwind CSS (industry standard) or UnoCSS (faster, more flexible)
- **Traditional framework?** → Bootstrap (most popular) or Bulma (modern Flexbox)
- **React animations?** → Motion/Framer Motion (declarative) or GSAP (complex timelines)
- **Simple scroll animations?** → AOS or ScrollReveal (just add attributes)
- **CSS-in-JS for React?** → Styled Components (popular) or Vanilla Extract (zero-runtime)
- **Tailwind companion tools?** → CVA + tailwind-merge + clsx (the standard trio)
- **Minimal styling?** → Pico CSS or Water.css (just works on semantic HTML)
