## Selection Criteria Applied
- Every repo above has 2,000+ GitHub stars (most have 8k+)
- All actively maintained (commits within 2025-2026)
- Self-hostable with clear deployment documentation
- Privacy-respecting where applicable
- Permissive or open-source licenses

## How to Choose
- **Simple, private analytics?** -> Plausible (simplest, beautiful) or Umami (self-hosted, clean)
- **Full GA replacement?** -> Matomo (most features) or PostHog (product analytics + more)
- **Product analytics (funnels, replays)?** -> PostHog (all-in-one)
- **SEO auditing?** -> Lighthouse (per page) or Unlighthouse (entire site)
- **Next.js SEO?** -> next-seo (meta tags, JSON-LD made easy)
- **Performance monitoring?** -> web-vitals (Core Web Vitals) + Partytown (offload scripts)
- **Image optimization?** -> Sharp (fastest, used by Next.js)
- **Error tracking?** -> Sentry (industry standard) or Highlight (all-in-one)
- **Bundle size control?** -> Size Limit (CI budget) + Bundle Analyzer (visualization)
