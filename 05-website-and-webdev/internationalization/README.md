# Internationalization & Localization

> Open-source i18n/l10n tools — translation management, locale formatting, RTL support, and multilingual content delivery for global websites.

---

## JavaScript i18n Frameworks

| Repository | Stars | Description |
|---|---|---|
| [i18next](https://github.com/i18next/i18next) | 8k+ | Most popular i18n framework. Plurals, interpolation, nesting, namespaces, plugins, all frameworks. |
| [next-intl](https://github.com/amannn/next-intl) | 3k+ | Internationalization for Next.js App Router. Type-safe, ICU messages, middleware routing. |
| [react-i18next](https://github.com/i18next/react-i18next) | 9k+ | React bindings for i18next. Hooks, HOC, render props, suspense, SSR support. |
| [vue-i18n](https://github.com/intlify/vue-i18n) | 2k+ | Internationalization for Vue 3. Composition API, message format, datetime, number. |
| [FormatJS (react-intl)](https://github.com/formatjs/formatjs) | 14k+ | ICU message format for JavaScript. Plurals, select, date/number formatting. React components. |
| [Lingui](https://github.com/lingui/js-lingui) | 5k+ | Lightweight i18n. ICU MessageFormat, macro-based extraction, React/Vue/Svelte. |
| [Paraglide.js](https://github.com/nicedoc/nicedoc.io) | 1k+ | Compiled i18n for SvelteKit/Next.js. Tree-shakeable, type-safe, no runtime overhead. |
| [typesafe-i18n](https://github.com/nicedoc/nicedoc.io) | 2k+ | Fully type-safe i18n. Auto-generated types, no runtime parsing, tiny bundle. |

## Why These Are Top-Tier

- **i18next** — Industry standard, 8k+ stars, works with every framework
- **next-intl** — Best Next.js App Router i18n, type-safe, middleware routing
- **Weblate** — Most complete open-source translation platform
- **LibreTranslate** — Free, self-hosted AI translation, no API keys
- **date-fns** — 35k+ stars, tree-shakeable date formatting with locales
- **Tolgee** — Modern translation management with in-context editing

## Implementation Guide

| Use Case | Recommended Stack |
|---|---|
| Next.js multilingual | next-intl + Tolgee (translation mgmt) |
| React SPA | react-i18next + i18next + ICU messages |
| Vue/Nuxt | vue-i18n + nuxt-i18n module |
| Static site | Astro i18n + content collections |
| AI translation | LibreTranslate (self-hosted) or DeepL API |
| Date/currency | date-fns + dinero.js + Intl API |
| RTL support | Tailwind logical properties + rtlcss |
| Translation workflow | Weblate + Git integration |
