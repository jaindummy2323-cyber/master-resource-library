## Why These Are Top-Tier

- **Hyperswitch** — Route payments across 50+ processors, avoid vendor lock-in, 13k+ stars
- **Lago** — Best open-source billing for usage-based SaaS, 7k+ stars
- **Kill Bill** — Enterprise-grade, handles complex billing scenarios at scale
- **Invoice Ninja** — Most complete self-hosted invoicing, 8k+ stars
- **BTCPay Server** — Zero-fee Bitcoin payments, true self-sovereignty
- **Stripe ecosystem** — Industry standard, best DX, most payment methods

## Implementation Guide

| Use Case | Recommended Stack |
|---|---|
| SaaS subscriptions | Stripe Billing + Lago (usage tracking) |
| One-time purchases | Stripe Checkout or Hyperswitch |
| Marketplace/platform | Stripe Connect + Hyperswitch |
| Invoicing clients | Invoice Ninja (self-hosted) |
| Digital products | Polar or Lemonsqueezy |
| Crypto payments | BTCPay Server |
| Multi-gateway failover | Hyperswitch (smart routing) |
| Tax compliance | Stripe Tax or Lago Tax |
