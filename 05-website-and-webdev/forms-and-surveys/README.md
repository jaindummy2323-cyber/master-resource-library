## Why These Are Top-Tier

- **Formbricks** — Most complete open-source survey platform, active development, self-hostable
- **React Hook Form** — Industry standard for React forms, 42k+ stars, minimal bundle
- **Zod** — TypeScript validation gold standard, used by Next.js, tRPC, and thousands of projects
- **Uppy** — Best-in-class file upload with plugin ecosystem and resumable uploads
- **SurveyJS** — Enterprise-grade survey library that works with any framework
- **Typebot** — Best conversational form builder, beautiful UI, great DX

## Getting Started

| Use Case | Recommended Stack |
|---|---|
| Simple contact form | React Hook Form + Zod + shadcn/ui |
| Customer surveys | Formbricks (self-hosted) |
| Conversational forms | Typebot |
| Enterprise data collection | SurveyJS + React JSONSchema Form |
| File-heavy forms | React Hook Form + Uppy |
| Multi-step onboarding | Formity + Zod + Tanstack Form |
