# 3D, WebGL & Interactive Experiences

> Open-source 3D rendering, WebGL/WebGPU frameworks, React Three Fiber ecosystem, shaders, physics engines, and tools for building immersive $20k-$50k web experiences.

---

## Core 3D Engines

| Repository | Stars | Description |
|---|---|---|
| [Three.js](https://github.com/mrdoob/three.js) | 103k+ | The standard for 3D on the web. WebGL/WebGPU renderer, massive ecosystem, examples library. |
| [Babylon.js](https://github.com/BabylonJS/Babylon.js) | 23k+ | Microsoft's 3D engine. PBR, physics, particles, GUI, XR support, playground editor. |
| [PlayCanvas](https://github.com/playcanvas/engine) | 10k+ | High-performance WebGL engine. Visual editor, physics, VR/AR, mobile-optimized. |
| [Godot (Web Export)](https://github.com/nicedoc/nicedoc.io) | 92k+ | Game engine with HTML5 export. Full 3D/2D, GDScript, C#, physics, shaders. |
| [A-Frame](https://github.com/aframevr/aframe) | 16k+ | Web VR/AR framework on top of Three.js. Declarative HTML, entity-component system. |
| [Filament](https://github.com/nicedoc/nicedoc.io) | 18k+ | Google's PBR rendering engine. Real-time global illumination, physically-based materials. |
