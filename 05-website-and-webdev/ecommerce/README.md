## Selection Criteria Applied
- Every repo above has 1,000+ GitHub stars (most have 5k+)
- All are actively maintained (commits within 2025-2026)
- Complete e-commerce solutions (not fragments)
- Self-hostable with clear deployment docs
- Permissive or open-source licenses

## How to Choose
- **Shopify alternative (headless)?** -> Medusa (Node.js, most popular) or Saleor (Python, GraphQL)
- **Laravel/PHP shop?** -> Bagisto (full-featured, multi-language)
- **Ruby on Rails shop?** -> Spree (flexible) or Solidus (enterprise fork)
- **Next.js storefront?** -> Vercel Commerce (works with any backend)
- **Payment processing?** -> Stripe (standard) or Hyperswitch (multi-processor routing)
- **Usage-based billing?** -> Lago (metering + invoicing)
- **Subscription billing?** -> Kill Bill (enterprise) or Lago (modern)
- **Multi-vendor marketplace?** -> Spree or Reaction Commerce
