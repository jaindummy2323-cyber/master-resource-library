# Website & Web Development

Curated collection of the best open-source tools for website building and web development.

## Subcategories

| Folder | What It Covers |
|--------|---------------|
| `landing-pages/` | High-converting landing page templates |
| `full-stack-starters/` | Complete app boilerplates (React, Next.js, etc.) |
| `no-code-lowcode/` | Builders, visual editors, drag-and-drop tools |
| `cms-tools/` | Content management systems, headless CMS |
| `design-systems/` | UI component libraries, design tokens |
| `ecommerce/` | Store builders, cart systems, payment integration |
| `portfolio-templates/` | Personal sites, agency sites, showcases |
