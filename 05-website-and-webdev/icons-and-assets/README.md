## Selection Criteria Applied
- Every repo has significant GitHub stars and community adoption
- All actively maintained with regular updates
- Free for commercial use (MIT, Apache 2.0, or similar)
- High quality, consistent design language
- Easy integration with modern web frameworks

## How to Choose
- **Modern React project?** -> Lucide (shadcn default) or Heroicons (Tailwind companion)
- **Largest free icon set?** -> Tabler Icons (5,900+) or Font Awesome Free
- **Brand logos?** -> Simple Icons (3,100+ brands)
- **One library for all icons?** -> Iconify (200k+ icons from 150+ sets)
- **Self-hosted fonts?** -> Fontsource (no Google tracking)
- **User avatars?** -> DiceBear (20+ styles) or Boring Avatars
- **Animated backgrounds?** -> tsParticles or Vanta.js
