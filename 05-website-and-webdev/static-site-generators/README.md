# Static Site Generators

> The best open-source static site generators for blogs, documentation, marketing sites, and high-performance web applications.

## JavaScript-Based SSGs

| Repo | Stars | Description | Why It's Top-Tier |
|------|-------|-------------|-------------------|
| [Astro](https://github.com/withastro/astro) | 48k+ | The web framework for content-driven websites. | Best performance. Islands architecture. Use any UI framework. |

## Selection Criteria Applied
- Every repo above has 5,000+ GitHub stars
- All are actively maintained (commits within 2025-2026)
- Production-ready with clear documentation
- Strong community and theme/plugin ecosystems
- Permissive licenses (MIT, Apache 2.0)

## How to Choose
- **Content/marketing site?** -> Astro (best performance, any framework) or Hugo (fastest builds)
- **Documentation site?** -> Docusaurus (Meta-built, versioning) or Starlight (Astro-powered, modern)
- **Blog?** -> Hugo (speed + themes), Ghost (if you want a CMS), or Astro (flexibility)
- **GitHub Pages?** -> Jekyll (native support) or Hugo (fast alternative)
- **Python ecosystem?** -> MkDocs + Material theme (beautiful docs)
- **React project docs?** -> Docusaurus or Nextra (Next.js-based)
- **Vue project docs?** -> VitePress (official Vue tool)
- **Minimum JavaScript output?** -> Eleventy (zero JS by default) or Astro (islands)
- **Single binary, no dependencies?** -> Hugo (Go) or Zola (Rust)
