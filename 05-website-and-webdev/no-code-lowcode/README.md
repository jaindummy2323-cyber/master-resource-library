## Selection Criteria Applied
- Every repo above has 2,000+ GitHub stars (most have 10k+)
- All actively maintained (commits within 2025-2026)
- Self-hostable with Docker or cloud options
- Clear documentation and onboarding
- Permissive or open-source licenses

## How to Choose
- **Internal tools (drag-and-drop)?** -> Appsmith or ToolJet (most integrations)
- **Internal tools (code-based)?** -> Refine (React framework, full control)
- **Airtable replacement?** -> NocoDB (50k+ stars) or Baserow (full-featured)
- **Zapier replacement?** -> n8n (most popular, 400+ integrations)
- **Survey/forms?** -> Formbricks (in-app surveys) or HeyForm (Typeform alternative)
- **Chatbot-style forms?** -> Typebot (visual flow builder)
- **Script-based workflows?** -> Windmill (Python/TS/Go scripts as workflows)
