# AI & LLM Tools

Curated collection of the best open-source AI and Large Language Model tools.

## Subcategories

| Folder | What It Covers |
|--------|---------------|
| `ai-agents/` | Autonomous agent frameworks, multi-agent systems |
| `prompt-engineering/` | Prompt libraries, optimization tools, testing frameworks |
| `rag-frameworks/` | Retrieval-augmented generation systems |
| `chatbot-builders/` | Conversational AI, customer-facing bots |
| `model-deployment/` | Serving, fine-tuning, inference optimization |
| `ai-wrappers/` | Tools built on top of LLMs for specific use cases |
| `voice-ai/` | Text-to-speech, speech-to-text, voice cloning |
| `image-ai/` | Image generation, editing, upscaling tools |
