## Recommended Stack (Start Here)

If building an agent from scratch, this is the proven stack:

| Layer | Tool |
|-------|------|
| Framework | LangGraph (stateful) or CrewAI (multi-agent teams) |
| Tool Protocol | MCP via mcp-agent + fastapi_mcp |
| Memory | Nocturne Memory or LangChain memory modules |
| Research | gpt-researcher for autonomous deep web research |
| Principles | 12-Factor Agents -- read before writing any code |
| Observability | LangSmith (LangChain) or Haystack tracing |
