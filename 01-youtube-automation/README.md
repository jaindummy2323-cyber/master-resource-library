# YouTube Automation

Curated collection of the best open-source tools for YouTube automation.

## Subcategories

| Folder | What It Covers |
|--------|---------------|
| `video-creation/` | AI video generators, text-to-video, auto-editing tools |
| `thumbnail-generation/` | AI thumbnail makers, template systems |
| `seo-and-analytics/` | YouTube SEO tools, keyword research, analytics scrapers |
| `content-repurposing/` | Long-to-short converters, clip extractors, multi-platform tools |
| `scheduling-and-publishing/` | Upload automation, scheduling tools |
| `captioning-and-subtitles/` | Auto-caption, translation, subtitle tools |
| `channel-management/` | Bulk operations, comment management, growth tools |

## Selection Criteria
- Stars: >500 minimum, >1000 preferred
- - Actively maintained (commits within 6 months)
  - - Complete start-to-finish solutions
    - - Clear documentation and setup guides
      - - Permissive license (MIT, Apache 2.0)
