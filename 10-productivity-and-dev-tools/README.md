# Productivity & Dev Tools

Curated collection of the best open-source productivity and developer tools.

## Subcategories

|--------|---------------|
| `cli-tools/` | Terminal utilities, shell scripts, workflow automation |
| `browser-extensions/` | Productivity extensions, automation |

## Selection Criteria
- Stars: >500 minimum, >1000 preferred
- Actively maintained (commits within 6 months)
- Complete start-to-finish solutions
- Clear documentation and setup guides
- Permissive license (MIT, Apache 2.0)
