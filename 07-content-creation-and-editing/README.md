# Content Creation & Editing

Curated collection of the best open-source tools for content creation and editing.

## Subcategories

| Folder | What It Covers |
|--------|---------------|
| `video-editing/` | Auto-editors, effects, transitions |
| `image-generation/` | AI art, photo editing, graphic design |

## Selection Criteria
- Stars: >500 minimum, >1000 preferred
- Actively maintained (commits within 6 months)
- Complete start-to-finish solutions
- Clear documentation and setup guides
- Permissive license (MIT, Apache 2.0)
